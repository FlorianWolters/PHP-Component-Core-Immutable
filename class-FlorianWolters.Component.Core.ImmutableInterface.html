Interface ImmutableInterface

The interface FlorianWolters\Component\Core\ImmutableInterface can be implemented by a class to mark it as immutable.

The FlorianWolters\Component\Core\ImmutableInterface has no methods or fields and serves only to identify the semantics of being immutable.

This interface is a Marker Interface that can be used to implement an Immutable Object.
