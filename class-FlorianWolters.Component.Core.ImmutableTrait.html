Trait ImmutableTrait

The trait FlorianWolters\Component\Core\ImmutableTrait helps to implement the Immutable Object implementation pattern.

The trait implements the following:

  • Avoids setting of inaccessible properties ($obj->value = 'foo).
  • Avoids calling the constructor on an object directly ($obj->__construct()).
